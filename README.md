# [A.T. RPG Game](https://polar-crag-98200.herokuapp.com/)

Our first real attempt at tackling jQuery and styling a live-hosted site.

## Basic Gameplay

Click a character to select as player. 
Pick which enemy to tackle first by clicking on the remaining characters.
Attack!
If you find your health depleting faster than your enemy's health, click the enemy to return him to the pool and select another.
The player's Attack Power will increase with each successful hit, so it is always possible to win!
